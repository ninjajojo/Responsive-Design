Structuring HTML

Semantic HTML5 Elements

Semantic tags, like HEADER and FOOTER, help make web pages clearer for both users and search engines, enhancing accessibility and SEO.

Using semantic tags improves accessibility for users with disabilities by assisting screen readers in navigating content. They also help search engines index and rank pages more effectively, improving site visibility.

Other benefits of semantic tags include:

  • code readability
  • SEO optimization
  • easier maintenance

CSS Selectors

Attribute selectors allow styling of elements based on attributes, such as targeting a span with a CLASS or styling input elements with a TYPE of SUBMIT.

For example, to style a span with a class of "highlight", you can use the selector span.highlight. This targets all span elements with the class "highlight".
